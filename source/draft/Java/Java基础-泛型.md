#### 泛型

> 在创建对象或调用方法的时候才明确下具体的类型

#### 泛型的好处

* 代码更加简洁(不需要强制转换)
* 程序更加健壮(在编译期间没有警告,在运行期就不会出现`ClassCastException`)
